### 1️⃣ Добавь разрешения в Info.plist (30 секунд)

**В Xcode:**
1. Открой `Info.plist`
2. Нажми `+` два раза
3. Добавь:

```
Privacy - Camera Usage Description
→ Доступ к камере

Privacy - Photo Library Additions Usage Description
→ Доступ к галерее
```

**Если Info.plist не открывается:**
1. Кликни на проект (вверху) → Target → вкладка Info
2. Добавь те же строки там
